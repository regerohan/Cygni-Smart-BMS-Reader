# Battery Monitor
A Battery Monitor for Cygni Batteries with SMART BMS

### What is this repository for? ###

This reports back battery parameters like SOC, Instantaneous current, Temperature etc.

### How do I get set up? ###

* Clone the repo
* Import Project in PlatformIO 
* Enjoy
